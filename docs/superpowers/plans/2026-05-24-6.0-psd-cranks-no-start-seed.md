**Research input:** `docs/superpowers/specs/2026-05-24-6.0-psd-cranks-no-start-research-input.md`

**Fact-check corrections to fold into seed content:**
1. Standpipe torque is **year-split**: 33 ft-lb on 2003–early-2004 log-rail trucks; **60 ft-lb on late-2004–2007 wavy-rail trucks**.
2. Fuel cranking pressure is **marked unresolved**: Ford service info says ≥45 psi all conditions; multiple aftermarket sources document 10–15 psi crank / 40–45 psi run as normal. Surface the disagreement rather than pretend it's settled.
3. Hot-restart Pattern #1 candidate is **injector top O-rings** (not STC fitting). The 99% quote attached to that row argues for #1, not #3.
4. Valley/HPOP cover access — **turbo removal IS required** (explicit), intake removal optional on '05–'07.
5. ICP-sensor-unplug PCM-substituted value is **uncertain** (750 psi most commonly cited, also 700, also 870).
6. Glow plug current draw: **~80 A per bank initial / 10–12 A per plug** (reconcile the two figures, don't conflate).
7. Oil cooler ΔT spec: **≥15°F oil-to-coolant at steady highway = restricted oil cooler** (replaces the "no verified source" marker).
8. FICM PIDs (FICM_VPWR / FICM_LPWR / FICM_MPWR) — keep, but soften the "no verified Ford source" disclaimer per Ford Doctors DTS PID list cross-reference.
